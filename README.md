# 500

Minimalist link dump for macOS.

## Features
- Drag & drop URLs from any browser
- Instant search
- Choose browser per link
- Quick rename (click to edit, double-click to open)
- Export to HTML Bookmarks, JSON, Markdown
- Adjustable font size (⌘+/⌘-)

## Install
1. Download `500.dmg`
2. Open DMG, drag 500 to Applications
3. Right-click → Open (first launch only)

## Requirements
- macOS 13.0+
